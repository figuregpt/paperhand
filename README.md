Practice trading Solana memecoins without risking real money.

**Live:** [paperhand.site](https://paperhand.site)

## Features

- **Paper Trading** - Trade with $10,000 virtual balance
- **Real-Time Prices** - Live token data from DexScreener
- **Slippage Simulation** - Realistic trading experience (0.5% - 5%)
- **Portfolio Tracking** - Track holdings, P&L, and transaction history
- **Shareable P&L Cards** - Generate and share your trading performance
- **Trending & New Tokens** - Discover popular and newly listed tokens
- **Favorites** - Save tokens to track (persists through reset)
- **Platform Detection** - Links to pump.fun, bonk.fun, Raydium, etc.
- **Mobile Responsive** - Works on all devices

## Tech Stack

- React + Vite
- Tailwind CSS
- Zustand (state management)
- Recharts (portfolio chart)
- html-to-image (P&L cards)
- DexScreener API

## Getting Started

```bash
# Clone the repo
git clone https://github.com/figuregpt/paperhand.git
cd paperhand

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## How It Works

1. Browse trending tokens or search by name/contract address
2. Select a token to view details and chart
3. Buy tokens with your virtual balance
4. Track your portfolio performance
5. Sell tokens and share your P&L cards
